# Baseball Archives 台本スタイルガイド

> Baseball inc.の文体を分析・抽出し、Baseball Archives用にアレンジした台本作成ガイドライン。
> 主な用途：8分前後の長尺「今週/今月のMLB日本人選手まとめ」動画。

---

## 1. 基本コンセプト

**野球解説ではなく、野球を題材にしたドキュメンタリーを作る。**

- 視聴者は「データを学びに来ている」のではなく「物語を体験しに来ている」
- 数字は物語の証拠として配置する。数字単体では飾らない
- ナレーターは「客観的な観察者」として静かに、しかし熱く語る

---

## 2. 必須要素（毎回必ず入れる）

### 2-1. 冒頭フック（0:00-0:30）
以下のいずれか1つで掴む：
- **衝撃の事実1行**：「○○が史上初の記録を達成した」
- **対比の予告**：「○○が栄光を掴んだ一方で、××は屈辱の底にいた」
- **印象的シーン描写**：「マウンドで帽子を投げ捨てて叫んだあの瞬間」
- **問いかけ**：「なぜ絶対王者だった日本は頂点を失ったのか」

### 2-2. 数字の組み込み
- 単独で「すごい」と言わない
- **必ず比較対象とセットで提示**
  - ❌「防御率1.85」
  - ✅「昨年4勝発敗と苦渋をなめた菅野が、今期は防御率1.85という別人のような成績」

### 2-3. キラーフレーズ
1動画につき**2〜3個**の強い言葉を埋め込む：
- 「異次元」「絶対王者」「神の右手」「正規の○○」
- 「○○の代償」「世界最強」「歴史の1ページ」
- 「○○が変わった瞬間」「運命の○○」

### 2-4. 対比表現
**Baseball inc.の最大の武器**。必ず1回以上：
- 「投手大谷/打者翔平」型（同一人物の二面性）
- 「日本が計算する中、メキシコはディズニーランドの待ち時間を計算していた」型（状況対比）
- 「前年4勝、今年15勝」型（時間軸対比）

### 2-5. 「絶望→希望」の振り子構造
動画の中盤に必ず配置：
1. 絶望シーン（不振・離脱・敗北）を丁寧に描写
2. 沈黙の時間（「このまま黙っているはずがありませんでした」型の溜め）
3. クライマックス（数字や具体シーンで描く）
4. 余韻（人物の言葉や反応で締める）

---

## 3. 文体ルール

### 3-1. 時制
- **基本は過去形**：「〜していました」「〜だったと言います」
- 名場面のみ現在形でリアリティ：「センターへ綺麗に弾き返した」

### 3-2. 主観表現
- ❌「すごい」「素晴らしい」「感動的」
- ✅ 数字 / 引用 / 周囲の反応で間接的に伝える
  - 「観客総立ちとなり」
  - 「アメリカメディアは『正気と思えない展開』と興奮しながら伝えました」

### 3-3. 引用の使い方
**本人の発言は必ずそのまま引用**：
- 大谷「憧れるのをやめましょう」
- 一郎「自分たちで育てていく」
- 引用後に解説を1〜2文添える

### 3-4. 比喩・ことわざ
シリアスすぎないよう、軽めの比喩を散りばめる：
- 「先行花火のような勢い」
- 「黒い雲の中へと姿を消しました」
- 「手のひら返しで手首がねじきれるような声」
- 「○○ありがとう」のような軽口（過剰に多用しない）

## 4. 構成テンプレート

### 4-1. 8分動画「今週のMLB日本人選手まとめ」用テンプレ

```
【0:00-0:30】イントロ（150字）
- 今週の象徴的1シーンで掴む
- 「今週、○○が△△を達成した。一方で...」と対比予告
- 動画全体のテーマを1行で示唆

【0:30-2:00】ドジャース3人衆（450字）
- 大谷の今週成績（数字＋逸話）
- 山本由伸の状態
- 佐々木朗希の課題への踏み込み
- 「投手大谷/打者翔平」など対比表現を配置

【2:00-3:30】カブス・レッドソックス組（450字）
- 鈴木誠也・今永の好調エピソード
- 吉田の物足りなさを数字で
- カブスの日本人2人の対比など

【3:30-5:00】メッツ・ロッキーズ・WS・BJ（450字）
- 千賀のIL状況（屈辱要素）
- 菅野・村上・岡本のNPB大物の現状
- 村上のホームラン量産があれば最大スポット

【5:00-6:30】アストロズ・エンゼルス（450字）
- 今井達也のIL状況
- 菊池雄星の現状
- マイナー選手の状況も触れる

【6:30-7:30】総括セクション（300字）
- 12選手のメジャー全体での位置づけ
- ランキング・賞争いへの言及
- 「数字で見る今週」型のまとめ

【7:30-8:00】クロージング（150字）
- 来週の見どころ
- キラーワードでオチ
- 次回への期待感を残す
```

### 4-2. 個人特集型テンプレ（応用編）

```
1. 衝撃の現在地（MVP獲得など）
2. ルーツ（高校時代・NPB時代）
3. 重大な選択（移籍・転換点）
4. 運命の出会い（コーチ・チームメイト）
5. ブレイクスルー（飛躍シーン）
6. 周辺ストーリー（他選手との対比）
7. 未来への布石
```

### 4-3. ペナント・優勝争い型テンプレ

```
1. 開幕前の予想（誰が本命だったか）
2. 春先の動き（フラグ立て）
3. 中盤の混戦
4. 救世主の登場
5. 終盤の天王山
6. 優勝決定の瞬間
7. 勝因の数字での総括
```

## 5. 「Baseball inc.メソッド」共通フロー

どのテーマでも、根底は同じ8段階：

1. **冒頭フック**（衝撃シーン or 結論の予告）
2. **背景説明**（誰が何を期待していたか）
3. **出来事の連鎖**（時系列）
4. **試練・絶望**（必ず入れる）
5. **救世主・転換点**（人物 or 出来事）
6. **クライマックス**（決定的瞬間）
7. **数字での総括**（前年比 or 歴代比較）
8. **余韻・次への布石**（次回への期待感）

---

## 6. 禁止事項

- ファンへの直接呼びかけ（「皆さん」「視聴者の皆様」）
- 弱い断定（「○○といえる」「○○ではないだろうか」）
- 抽象的形容詞のみで終わる文（「素晴らしい活躍でした」だけで終わる）
- データの羅列のみで物語性なし
- 主観的感情の押し付け（「感動的でした」）
- 同じキラーワードの繰り返し使用

## 8. キラーフレーズ集（ストック）

### 投手系
- 「マウンドを支配した」
- 「○球を投げた、その全てが伝説となった」
- 「絶対のクローザーが沈黙した夜」
- 「日本人投手陣、戦慄の現実」

### 打者系
- 「バットが沈黙する」
- 「歴史を変える1打」
- 「神の○○」（○○には部位や行動）
- 「数字が嘘をついている」

### 状況系
- 「○○が変わった瞬間」
- 「運命の○○」
- 「正規の○○」（不名誉な事実を強調）
- 「○○の代償」

### 結論系
- 「これが二刀流の代償だ」
- 「全米震撼」
- 「歴史の1ページに刻まれた」
- 「○○年越しの○○」

## 9. 利用方法

### 台本作成時のClaude指示例
```
今週分の大谷翔平のデータで、Baseball inc.スタイルの台本を作って。
style_guide.mdに従い、特に以下を意識：
- 冒頭フックは「絶望→希望」型で
- キラーフレーズは2-3個使用
- 8分動画用テンプレに沿って構成
- 数字は必ず比較対象とセットで
```

### 改善ループ
1. 動画公開後、再生数・コメントを観察
2. うまくいった表現は本ファイルの「キラーフレーズ集」に追加
3. 反響悪かった構成パターンは「禁止事項」に追加
4. 月1回、ガイドを更新
